# -Unit-Converter-
Outil interactif pour convertir différentes unités de longueur (mètres, kilomètres, centimètres). L’utilisateur saisit une valeur et obtient instantanément la conversion dans l’unité souhaitée. Ce projet m’a permis de pratiquer la logique JavaScript, la gestion des événements et la manipulation du DOM.  ⚙️ Technologies : HTML • CSS • JavaScript
